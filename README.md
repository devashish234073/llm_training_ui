# llm_training_ui

Steps to setup and launch

```
1. git clone https://github.com/devashish234073/llm_training_ui
2. cd llm_training_ui
3. pip install -r requirements.txt
4. python app.py
5. visit http://127.0.0.1:5000/ in browser
```
